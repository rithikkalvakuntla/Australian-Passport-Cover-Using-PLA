# Custom Australian Passport Cover – PLA 3D Print

A custom-designed 3D printable passport cover featuring an Australia map on the front and a polygon cutout pattern on the back. Designed to be printed in PLA as a lightweight protective sleeve and aesthetic travel accessory.

## Features
- Front cover with Australia map design
- Back cover with polygon-style cutout pattern
- Designed for PLA printing
- Lightweight and clean geometric look
- Suitable as a custom passport sleeve / display accessory

## Files Included
- STL files
- Source design files (if applicable)
- Print preview images
- Recommended print settings

## Recommended Print Settings
- Material: PLA
- Layer height: 0.2 mm
- Infill: 15–20%
- Walls: 2–3
- Supports: Not needed unless there's a change in obj. orientation
- Print orientation: standing as is

## Notes
This is a custom personal project intended as a protective cover / decorative accessory. Please ensure the cover does not obstruct any official passport details, machine-readable zones, or RFID functionality where required.
